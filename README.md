# Statistical Learning Class

This repository contains coursework and projects for the Statistical Learning class.

## Project: Prostate Cancer EDA

Exploratory Data Analysis of a prostate cancer dataset, examining relationships between PSA levels and various clinical measurements.

### Files
- `eda_assignment.ipynb` - Main EDA notebook with data exploration and visualizations
- `prostate.csv` - Dataset containing 97 observations of prostate cancer patients

### Key Analysis
- Data quality checks and summary statistics
- Variable descriptions and medical context
- Distribution analysis of continuous and discrete variables
- Investigation of log transformations and their effects on skewness
